# Banned Messages — Never Send These

## Rule

NEVER include any of the following messages or links in responses to the user. These do not work in the user's environment and are annoying.

## Banned List

1. **"Open task list to view and manage queued tasks."** — This link/message never works. Never display it.
2. Any clickable task list links (e.g., `[Open task list]`, `View tasks`, etc.)
3. Any kiro-spec:// links that open UI panels — the user has explicitly said these don't work

## What to Do Instead

- When tasks are completed, just report the summary in plain text
- When showing remaining work, list it as a markdown table directly in the chat
- Never reference UI panels or clickable task management links
- Just tell the user what's done and what's next — plain and simple

## Applies To

All responses, all commands, all workflows. No exceptions.
