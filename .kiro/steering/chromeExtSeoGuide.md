# Chrome Extension SEO Guide — Web Store Optimization

Standards for Chrome Web Store listing optimization (CWS ASO).

## Title Rules
- Clear and descriptive — users understand function from title alone
- Concise — shorter is more memorable
- Unique — avoid similarity to existing extensions
- NO keyword stuffing

## Summary (132 chars max)
- Lead with user BENEFIT not feature
- Highlight main use case
- Include 1-2 keywords naturally
- No superlatives ("best", "fastest", "greatest")
- No competitor references

## Description
- Opening paragraph: what it does + who it's for
- Feature bullet list (5-8 items)
- "How to use" steps (numbered)
- Privacy statement
- Keywords woven naturally — NO repetitive keyword spam

## Icon (128×128 + 48×48)
- Simple, recognizable at 16px (toolbar size)
- Consistent brand colors
- No screenshots or UI elements in icon
- No text (unreadable at small sizes)

## Screenshots (1280×800 or 640×400, up to 5)
- MINIMUM 3 screenshots required for quality listing:
  1. Idle/default state
  2. Success state (green indicators)
  3. Active/running state
  4. Error handling state (optional but good)
  5. Populated data state (optional)
- Full bleed, square corners, no padding
- Clear, not blurry or distorted
- Show actual UI, not mockups
- Include visual aids/annotations if helpful
- Consistent branding across all screenshots

## Promo Tiles
- **Small (440×280):** Simple, saturated colors, minimal text, fill entire region
- **Marquee (1400×560):** Uncluttered, high-res, consistent branding
- Avoid too much text
- Avoid white/light gray backgrounds
- Make sure edges are well defined
- Works at half size

## Required Dashboard Fields
- [ ] Website URL (your main site)
- [ ] Support URL (bug report page)
- [ ] Privacy policy URL
- [ ] Categories selected
- [ ] Contact email verified

## Quality Signals (affects ranking)
- High ratings (aim for 5 reviews minimum to show stars)
- Low uninstall rate
- Regular updates
- Good permissions justification
- Fast, responsive UI
- Clear single purpose

## Checklist Before Publish
- [ ] Title: clear, concise, unique
- [ ] Summary: benefit-led, <132 chars
- [ ] Description: paragraph + bullets + how-to
- [ ] Icon: simple at 16px, 48px, 128px
- [ ] Screenshots: 3-5, clear, branded
- [ ] Promo tiles: small + marquee
- [ ] Website URL set
- [ ] Support URL set
- [ ] Privacy policy URL set
- [ ] Permissions justified
- [ ] Single purpose described
- [ ] No keyword spam
- [ ] Version number updated
