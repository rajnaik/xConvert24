## Full Description

**Auto Clicker — Schedule Clicks on Any Web Element**

Automatically click any button, link, or element on a web page at a regular interval. Perfect for:

- Refreshing dashboards and monitoring pages
- Keeping sessions alive
- Automating repetitive clicks
- Testing web applications
- Auto-refreshing feeds or data tables

**Features:**

- Target any element using #ID or CSS selectors
- Set click interval from 1 to 60 seconds
- Test button verifies the element exists before starting
- Visual countdown shows time until next click
- Green/red status indicators for success/failure
- Saves your selectors for quick reuse
- Works on any website
- Minimal permissions — only accesses the active tab
- No data collection, no tracking, fully private

**How to Use:**

1. Navigate to the page with the element you want to auto-click
2. Click the extension icon
3. Enter the CSS selector (e.g., #refresh-btn, .load-more, [data-action="next"])
4. Set your interval (1-60 seconds)
5. Click "Test" to verify it works
6. Click "Start" — the extension clicks automatically at your interval

**Privacy:**
All settings stored locally. No data leaves your browser. No analytics. No tracking.
